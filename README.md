# 🍽️ Harisenin E-Menus App

**Harisenin E-Menus** adalah aplikasi manajemen menu digital berbasis web yang dikembangkan menggunakan **React** dan **Vite**. Proyek ini merupakan bagian dari kurikulum Fullstack Web Development di **Harisenin.com** untuk mendemonstrasikan penguasaan operasi CRUD dan integrasi API modern.

## 🚀 Fitur Utama
* **Manajemen Menu (CRUD)**: Menambah, melihat detail, memperbarui, dan menghapus data menu secara real-time.
* **State Management Kompleks**: Implementasi `useReducer` untuk menangani logika formulir yang dinamis dan terstruktur.
* **Global Theme Management**: Fitur **Dark Mode** yang dikelola menggunakan library `Zustand`.
* **Integrasi API**: Penggunaan `Axios` untuk pengambilan data asinkron dari MockAPI, lengkap dengan penanganan status *loading* dan *error*.
* **Routing Dinamis**: Navigasi antar halaman (Dashboard, Detail, Edit) menggunakan `React Router`.

## 🛠️ Tech Stack
* **Frontend**: React.js (Vite)
* **State Management**: Zustand & useReducer
* **API Client**: Axios
* **Styling**: Modern CSS dengan CSS Variables
* **Backend Simulator**: MockAPI
